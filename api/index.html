Prever idade pelo nome (Agify)

Digite seu nome abaixo. O site enviará o nome para a API e mostrará a previsão de sua idade.

Como funciona

Ao enviar o nome, é feito uma chamada para Agify. A resposta contém a idade estimada e a contagem de registros usados pela API.
